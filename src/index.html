About the System

An academic project demonstrating cloud-native architecture for managing attendance at large-scale events.

  • EC2 Auto Scaling Groups for high availability
  • Application Load Balancer for traffic distribution
  • AWS Lambda for serverless computing
  • Amazon Aurora with Multi-AZ deployment
